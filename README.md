# Student Record Management System

## Project Description
Student Record Management System is a CLI-based Java application used to manage student information.
This system allows users to perform CRUD operations like adding, viewing, updating, and deleting student records.

## Features
- Add new student records
- View all student records
- Update existing student details
- Delete student records
- Menu-driven command line interface

## Technologies Used
- Java
- VS Code / IntelliJ IDEA

## Concepts Implemented
- Classes and Objects
- Encapsulation
- Constructors
- Getters and Setters
- ArrayList Collection
- Loops and Conditional Statements
- CRUD Operations
